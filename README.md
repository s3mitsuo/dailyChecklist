日常の物事の実行有無を記録するアプリを作成したものです。
Claudeツールを使用し、以下のプロンプトを入力することで、本アプリを作成しました。

# 日常の物事の実行有無を記録するアプリ

## 機能概要
日常の物事の実行有無を記録するアプリを作成する。

## 機能要件
- 各項目にチェックをつけられるようにする。
   - 初期項目は以下の3つとする。
      - 鍵の施錠
      - 窓の施錠
      - 部屋の消灯
- チェックをつけた日時を記録して表示する。
- チェックを外した場合は、日時の記録と表示を消去する。
- 再度チェックをつけた場合は、新たに日時を記録して表示する。
- 初期表示の項目だけでなく、新しい項目も追加できる。
- ブラウザのローカルストレージに内容を記憶できる。

## 実装方法
これらの機能を、シンプルなHTMLファイルとして実装する。
